# Privacy Notes

Synapse is being designed as a self-hosted research context layer. The initial defaults in this repository follow the Notion direction:

- no telemetry by default
- local-first storage in PostgreSQL and MinIO
- explicit configuration for any external LLM fallback
- research artifacts stay under user-controlled infrastructure unless the operator chooses otherwise

This file is a placeholder for the fuller GDPR-ready policy planned in the pre-launch checklist.
